# 文件搜索工具打包说明

## 打包状态

由于当前环境限制，electron-builder 无法直接完成打包过程。请按照以下步骤手动完成：

## 手动打包步骤

### 1. 打开命令提示符 (CMD)

在项目目录下打开终端：

```cmd
cd /d D:\SoloApp
```

### 2. 运行打包命令

选择以下任一打包方式：

#### 方式 A：便携版（推荐）
生成单个 exe 文件，可直接运行，无需安装

```cmd
npm run electron:portable
```

生成文件位置：`D:\SoloAppEnd\文件搜索工具-Portable.exe`

#### 方式 B：解压版
生成文件夹，可直接运行

```cmd
npm run electron:build
```

生成文件位置：`nd\win-unpacked\文件搜索工具.exe`

#### 方式 C：安装版
生成安装程序，可安装到系统

```cmd
npm run electron:dist
```

生成文件位置：`D:\SoloAppEnd\win-unpacked\ 文件夹内`

## 快速打包脚本

直接双击项目中的 `打包.bat` 文件运行打包

## 打包后的文件

- **portable**：单个 exe 文件，直接运行
- **win-unpacked**：包含 exe 和运行所需文件的文件夹
- **win-unpacked**：安装程序文件

## 部署到其他电脑

1. 将打包生成的 `exe` 文件或整个 `win-unpacked` 文件夹复制到目标电脑
2. 直接运行即可，无需安装额外软件
3. 应用程序会独立运行，搜索本地磁盘文件
