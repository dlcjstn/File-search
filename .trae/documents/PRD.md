# 文件搜索工具 - 产品需求文档

## 1. 产品概述

**文件搜索工具**是一款高效的本地文件检索应用，灵感来源于 Everything，支持快速搜索 Windows 系统中的文件。应用采用现代化设计风格，提供直观友好的用户体验。

**目标用户**：需要在电脑上快速定位文件的专业用户，包括办公人员、开发者、设计师等。

**核心价值**：快速、准确、美观的文件搜索体验，支持多种筛选条件和视图模式。

---

## 2. 核心功能模块

### 2.1 功能总览

1. **关键词搜索** - 按文件名或后缀名搜索
2. **文件类型筛选** - 文档、图片、代码、压缩包等多种类型
3. **自定义后缀** - 添加用户自定义的文件扩展名
4. **双视图模式** - 列表视图和树状视图
5. **快速操作** - 双击打开文件或文件夹
6. **Everything集成** - 调用 Everything 加速搜索（可选）

---

## 3. 页面详细设计

### 3.1 整体布局

- **容器最大宽度**：1280px
- **间距系统**：基于8px的倍数
- **卡片式布局**：圆角16px，轻微阴影
- **背景**：渐变色背景（灰色系）

### 3.2 搜索区域设计

**设计风格**：极简现代，融合玻璃拟态效果

**组件构成**：
- 搜索输入框：大尺寸，聚焦时有明显反馈
- 搜索类型切换：单选按钮组
- 文件类型选择器：下拉菜单，支持多选标签
- 自定义后缀输入：内联输入框，带验证反馈
- 搜索按钮：渐变背景，悬停放大效果

**交互细节**：
- 输入框聚焦时边框发光效果
- 按钮悬停时缩放1.05倍
- 下拉菜单平滑展开/收起
- 标签按钮点击时有选中状态

### 3.3 结果展示区域

**列表视图**：
- 显示：文件名、路径、大小、修改日期
- 行悬停高亮
- 双击打开文件/文件夹
- 路径可单独点击打开文件夹

**树状视图**：
- 可折叠/展开文件夹
- 层级缩进显示
- 展开动画

**状态设计**：
- 空状态：友好提示文字
- 加载状态：旋转加载图标 + 进度提示
- 错误状态：红色警示框 + 错误信息
- 成功状态：结果计数 + 搜索耗时

### 3.4 视图切换

- 列表/树状视图切换按钮
- 当前视图高亮显示
- 平滑过渡效果

## 4. 设计规范

### 4.1 色彩系统

**主色调**：
- 深空蓝 `#1E3A5F` - 专业感
- 强调蓝 `#3B82F6` - 主要交互

**功能色**：
- 成功绿 `#10B981`
- 警告橙 `#F59E0B`
- 错误红 `#EF4444`

**中性色**：
- 背景渐变 `gray-50 → gray-100`
- 卡片白 `white`
- 文字灰 `gray-900/600/500`
- 边框灰 `gray-200`

### 4.2 字体规范

- **标题**：系统字体栈（SF Pro Display / Segoe UI）
- **正文**：系统字体栈
- **字号层级**：
  - H1: 2rem (32px)
  - H2: 1.5rem (24px)
  - Body: 1rem (16px)
  - Small: 0.875rem (14px)
  - Caption: 0.75rem (12px)

### 4.3 按钮样式

**主按钮**：
- 圆角12px
- 蓝色渐变背景
- 悬停：放大1.05倍 + 阴影加深
- 点击：缩小0.95倍

**次按钮**：
- 透明背景
- 灰色边框
- 悬停：填充背景

**图标按钮**：
- 48px × 48px 触控区域
- 悬停旋转或缩放

### 4.4 动效设计

**微交互**：
- 按钮点击：scale(0.95 → 1)，150ms ease-out
- 卡片悬停：translateY(-2px) + shadow增强，200ms
- 下拉展开：高度动画 + opacity，300ms
- 加载动画：旋转 + 脉冲，1.5s循环

**过渡效果**：
- 视图切换：淡入淡出，300ms
- 结果列表：交错进入动画

## 5. 技术实现

### 5.1 技术栈

- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式方案**：Tailwind CSS
- **图标**：Lucide React
- **状态管理**：React useState/useCallback
- **后端**：Express.js（可选）

### 5.2 组件结构

```
FileSearch
├── SearchInput（搜索输入框）
├── SearchTypeSelector（搜索类型切换）
├── ExtensionSelector（文件类型选择器）
├── CustomExtensionInput（自定义后缀）
├── SearchButton（搜索按钮）
├── ViewModeToggle（视图切换）
├── SearchResults（结果容器）
│   ├── ListView（列表视图）
│   └── TreeView（树状视图）
└── StatusBar（状态栏）
```
