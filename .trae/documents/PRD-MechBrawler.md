# 像素风机甲对战游戏 - 产品需求文档

## 1. 产品概述

一款复古像素风格的2D机甲对战小游戏，两名玩家控制各自的机甲进行战斗。游戏采用经典街机格斗游戏的操作模式，强调操作技巧与策略对抗。目标用户为喜欢复古游戏和机甲题材的玩家。

## 2. 核心功能

### 2.1 游戏角色

**机甲A (蓝色)**
- 颜色：科技蓝 + 银白色
- 特征：敏捷型机甲，移动速度快，攻击力适中
- 特殊技能：快速突进

**机甲B (红色)**
- 颜色：烈焰红 + 暗黑色
- 特征：力量型机甲，移动速度慢，攻击力高
- 特殊技能：强力重击

### 2.2 操作机制

**基本移动**
- 左右移动：键盘A/D 或 ←/→
- 跳跃：键盘W 或 ↑
- 防御：按住S 或 ↓

**攻击系统**
- 普通攻击：J键 - 快速拳击
- 重攻击：K键 - 强力击打
- 特殊技能：L键 - 机甲专属技能

**战斗规则**
- 血量系统：每台机甲100点HP
- 伤害数值：普通攻击10点，重攻击20点，特殊技能25点
- 防御减伤：受到攻击时按住防御键减少50%伤害
- 胜负判定：先击倒对方血量归零者获胜

### 2.3 游戏界面

**开始界面**
- 游戏标题（复古霓虹灯风格）
- "开始战斗" 按钮
- 操作说明面板

**对战界面**
- 战斗场景（工业废墟背景）
- 双方机甲角色
- 血量条（顶部显示）
- 能量条（释放特殊技能用）
- 操作提示（底部显示）

**结束界面**
- 胜负结果显示
- 重新开始按钮
- 返回主菜单按钮

## 3. 核心流程

### 3.1 游戏主流程

```
[开始界面] → 点击开始 → [加载动画] → [对战界面] → [战斗开始]
                                        ↓
                              [一方血量归零]
                                        ↓
                               [结束界面] → [重新开始/返回主菜单]
```

### 3.2 战斗流程

```
[待机状态] → 按下攻击键 → [攻击动画播放] → [碰撞检测] → [伤害判定] → [扣血动画] → [返回待机]
                                    ↓
                              [防御中] → [减伤50%]
```

## 4. 视觉设计

### 4.1 美术风格
- 像素密度：16x16像素基础单元
- 调色板：限制为32色复古调色板
- 动画帧率：8-12 FPS（经典像素动画节奏）
- 特效：粒子效果、屏幕震动、闪光

### 4.2 配色方案
- 背景：深灰 #1a1a2e、#16213e
- 机甲A：#00d9ff（科技蓝）、#c0c0c0（银白）
- 机甲B：#ff4757（烈焰红）、#2f3542（暗黑）
- UI元素：#ffd93d（金黄）、#6c5ce7（紫色霓虹）
- 血量条：#00ff88（绿色）→ #ff6b6b（红色渐变）

### 4.3 字体
- 主标题：Press Start 2P（像素字体）
- UI文字：VT323（复古终端字体）

### 4.4 动画设计
- 待机动画：呼吸效果，4帧循环
- 移动动画：行走周期，6帧循环
- 攻击动画：出拳/重击，3-5帧
- 受伤动画：后仰闪烁，3帧
- 胜利动画：机甲站立庆祝

## 5. 音效设计（可选）
- 攻击音效：机械打击声
- 受伤音效：金属碰撞声
- 胜利音效：欢呼胜利号角
- BGM：8-bit电子舞曲风格

## 6. 技术实现

### 6.1 渲染技术
- Canvas 2D 渲染
- 60 FPS 游戏循环
- 像素完美渲染（禁用抗锯齿）

### 6.2 物理系统
- 重力模拟：0.5 单位/帧²
- 移动速度：5 单位/帧
- 跳跃初速度：-12 单位/帧
- 碰撞盒：简化矩形检测

### 6.3 状态机
```
IDLE ←→ WALKING
  ↓        ↓
JUMPING ←┘
  ↓
ATTACKING → HIT_STUN → IDLE
  ↓
BLOCKING → IDLE
```

## 7. 素材方案

### 7.1 精灵图设计
- 机甲精灵：16x16基础，4方向，6种状态
- UI元素：血条、按钮使用CSS绘制
- 背景：Canvas绘制的静态场景

### 7.2 粒子效果
- 攻击火花：黄色/橙色粒子
- 防御护盾：蓝色光环
- 受伤出血：红色液滴

## 8. 游戏平衡性

### 8.1 数值设计
| 属性 | 机甲A | 机甲B |
|------|-------|-------|
| HP | 100 | 100 |
| 移动速度 | 5 | 4 |
| 普通攻击伤害 | 10 | 12 |
| 重攻击伤害 | 18 | 22 |
| 特殊技能伤害 | 25 | 30 |
| 防御减伤率 | 50% | 50% |
| 能量恢复速度 | 1/帧 | 1.2/帧 |

### 8.2 公平性保障
- 双方操作对称
- 伤害数值经过平衡调整
- 无随机暴击系统
- 特殊技能有冷却时间（3秒）
